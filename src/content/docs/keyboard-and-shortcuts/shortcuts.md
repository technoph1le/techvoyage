## **Keyboard Shortcuts Cheat Sheet**

### **Basic Editing**

* **Ctrl + C** — Copy
* **Ctrl + V** — Paste
* **Ctrl + X** — Cut
* **Ctrl + Z** — Undo
* **Ctrl + Y** — Redo
* **Ctrl + A** — Select all
* **Shift + Delete** — Delete permanently (bypass Recycle Bin)
* **F2** — Rename selected file/folder

### **File / Window Management**

* **Ctrl + N** — New window/document (varies by app)
* **Ctrl + S** — Save
* **Ctrl + P** — Print
* **Ctrl + F** — Find/search within a page
* **Ctrl + T** — New tab
* **Ctrl + Shift + T** — Reopen last closed tab
* **Ctrl + Shift + N** — New incognito/private window

### **System / OS Actions**

* **Ctrl + Alt + Delete** — Open security options (Task Manager, lock, etc.)
* **Ctrl + Shift + Esc** — Open Task Manager directly
* **Alt + F4** — Close current window/app
* **Windows + L** — Lock your computer
* **Windows + D** — Show desktop / minimize all windows
* **Windows + R** — Open Run dialog
* **Windows + E** — Open File Explorer

### **Navigation / Windows**

* **Alt + Tab** — Switch between open apps
* **Alt + Enter** — Show properties of selected item / fullscreen in some apps
* **Alt + Space** — Open window control menu (restore, minimize, etc.)
* **F11** — Toggle fullscreen mode
* **F5** — Refresh

### **Screenshots**

* **Print Screen** — Capture full screen to clipboard
* **Alt + PrtSc** — Capture active window to clipboard
